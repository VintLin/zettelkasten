# Structure Note Rules

## Goal
Use structure notes as entry points and navigation maps.

## When to create a structure note

Create or update a structure note when:
- 3 or more related permanent notes now form a meaningful area
- a topic needs guided traversal
- a project or essay needs an outline
- multiple notes are easier to use as a map than through raw backlinks alone

## When not to create one

Do not create a structure note when:
- there is only one isolated note
- the topic is still fuzzy
- the structure would only duplicate tags or folder names

## Functions of a structure note

A structure note can be:
- a topic entry point
- a map of a concept area
- a research dashboard
- a writing outline
- a comparison canvas

## Internal structure

Each structure note should contain:
- purpose
- entry points
- grouped subtopics
- optional open questions
- optional next writing moves

## Maintenance rule

Do not turn structure notes into giant dumps.
They should help the reader enter an area, not replace the permanent notes themselves.
